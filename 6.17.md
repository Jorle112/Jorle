## 辗转相除法

### 适用于求最大公约数

```cpp
int a = x; //保存原始数据
int b = y;
int r,divisor,multiple;

while(y != 0){ //循环直到余数为0
    r = x % y;
    x = y;
    y = r;
}

divisor = x;
multiple = a * b / divisor; // 最小公倍数=两数相乘除以最大公约数
```
## 判断闰年

bool isright(int year){
    return(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0); //能被4整除且不能被100整除，或者能被400整除
}
