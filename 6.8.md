## 冒泡排序

```cpp
for(int i = 0;i < n;i++){  //n为要排序的个数
        for(int j = 0;j < n-i;j++){
            if(a[j] < a[j+1])
            swap(a[j],a[j+1]);
        }
    }
```
## 折半查找

```cpp
//此示例为从左到右减小的查找
int left = 0,right = n;
int mid;
int found = 0;

while(left <= right){
        mid = (left + right) / 2;
        if(a[mid] == key){ // key为要查找的数
            found = 1; 
            break;
        }else if(a[mid] > key){
            left = mid + 1; //往右边小的地方找
        }else{
            right = mid - 1;
        }
    }
```
